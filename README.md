# Garbage Detection Web App

## 📌 Project Overview
The Garbage Detection Web App is a Computer Vision project that detects different types of garbage in images using a trained machine learning model.  
This project provides an interactive web interface to help users understand garbage detection and promote proper waste management.

---

## 🎯 Project Objectives
- Detect garbage in images using Computer Vision  
- Classify detected garbage into categories (e.g., plastic, paper, organic)  
- Provide a user-friendly web interface  
- Help users understand and manage waste better  

---

## ⚙️ Features
- Upload images through the web app  
- Detect garbage in the image using a trained ML model  
- Highlight detected garbage areas on the image  
- Interactive and simple UI for learning and demonstration  

---

## 🧠 Technologies Used
- Python  
- Flask (Backend)  
- HTML, CSS, JavaScript (Frontend)  
- OpenCV / Computer Vision libraries  
- Pre-trained ML model for garbage detection  
- VS Code (Development Environment)
